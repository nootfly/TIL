# SilverBullet Guide

SilverBullet is an open-source, local-first markdown note-taking tool that works with a "space" (a folder of markdown files).

## Key Features
- **Programmable Notes**: Use Lua to query and transform your notes.
- **Slash Commands**: Quick access to templates and tools.
- **Self-Hosted**: Can be run via Docker or as a standalone binary.

## Local Configuration
If you're running SilverBullet via Docker, your notes are typically mounted to `/space`.
